# Smart-To-Do-App

The Smart To-Do App is a simple and efficient task management web application built using HTML, CSS, and JavaScript. It allows users to add, manage, and delete daily tasks easily. The app uses Browser Local Storage to save tasks, ensuring that data is not lost even after refreshing or reopening the browser.

This project demonstrates core front-end development concepts and is suitable for beginners learning JavaScript and DOM manipulation.


 Features

Add new tasks dynamically

Delete tasks when completed

Tasks are saved using Local Storage

Data persists even after page refresh

Clean and user-friendly interface


Tech Stack

HTML – Structure of the application

CSS – Styling and layout

JavaScript – Functionality and logic

Local Storage – Persistent data storage


How It Works

User enters a task and adds it to the list

JavaScript stores the task in localStorage

On page reload, tasks are fetched from localStorage and displayed

Users can remove tasks, which updates localStorage in real time
